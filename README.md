🚗 Car Rental System

A modern web-based Car Rental System that allows users to browse, book, and manage car rentals بسهولة. The platform also provides admin features to manage vehicles, bookings, and users efficiently.
📌 Features
🔐 User Authentication (Login/Register)
🚘 Browse available cars
📅 Book cars with date selection
💳 Booking management system
📊 Admin dashboard (manage cars, users, bookings)
📱 Responsive UI for all devices
🔍 Search and filter cars

🛠️ Tech Stack

Frontend:

React 

Backend:

Node.js / Express 

Database:

MongoDB 

Other Tools:

Git & GitHub
REST APIs

⚙️ Installation & Setup

1️⃣ Clone the repository

   git clone https://github.com/shahidamzad/CarRental.git
   cd CarRental
   
2️⃣ Install dependencies
   npm install

3️⃣ Setup environment variables

PORT= 
MONGO_URI= 

JWT_SECRET =

IMAGEKIT_PUBLIC_KEY =
IMAGEKIT_PRIVATE_KEY =
IMAGEKIT_URL_ENDPOINT =

4️⃣ Run the project
   npm run dev

🚀 Usage
Register/Login as a user
Browse available cars
Select a car and book
View booking history

Admin can:

Add/Edit/Delete cars
Manage bookings
Manage users
